# DateInput

<ImportPath component={DateInput} section="forms" />

Component for inputting a single date. The `input` element will accept date entry in the
"MM/DD/YYY" format, or the user can use the calendar popover.

The display/input format of the date is not localized - it will always use the American
English "MM/DD/YYYY" format.

## Default selected date

The input element in `DateInput` will be empty by default.
Pass an instance of `Date` or a parsable date string (`MM/DD/YYYY`) to the `defaultDate` prop to set a default value.

      defaultDate={new Date()}
    />
  </Story>
</Preview>
