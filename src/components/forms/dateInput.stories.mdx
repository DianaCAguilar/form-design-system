# DateInput

Component for inputting a single date. The `input` element will accept date entry in the
"MM/DD/YYY" format, or the user can use the calendar popover.

<Story name="Knobs" parameters={{ docs: { disable: true }}}>
  <DateInput
    onDateChange={action('change')}
    futureYears={number('Future years', 5)}
    pastYears={number('Past years', 10)}
  />
</Story>

<Preview isExpanded>
  <DateInput onDateChange={(date) => {}} />
</Preview>

## Default selected date

Pass an instance of `Date` to the `defaultDate` prop to set a default value.

<Preview>
  <Story name="Default selected date">
    <DateInput
      onDateChange={action('change')}
      defaultDate={new Date()}
    />
  </Story>
</Preview>
