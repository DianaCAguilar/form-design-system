# InputGroup

<ImportPath component={InputGroup} section="forms" />

Visually groups multiple form fields on one line.
This component will automatically assign `shrink`/`grow` to its internal flex items, but
you can override the defaults by using the `flexSettings` prop.

<Props of={InputGroup} />

### Custom grow/shirnk

Use the `flexSettings` prop to control the flex behavior of each input segment.
In this example, the first and last inputs are set to `grow` and the middle input will
`shrink` to the natural width of the select.

<Preview>
  <Story name="Custom grow/shrink">
    <InputGroup flexSettings={['grow', 'shrink', 'grow']}>
      <DecoratedInput before="$">
        <input type="number" defaultValue="100" />
      </DecoratedInput>
      <select name="unit">
        <option value="1">Million</option>
        <option value="2">Thousand</option>
      </select>
      <input type="text" placeholder="Investor Name" />
    </InputGroup>
  </Story>
</Preview>
