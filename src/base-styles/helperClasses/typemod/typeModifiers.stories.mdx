## Color Modifiers
Modifies color of text.
When using standard text color classes (`heading`, `primary`, `secondary`, `hint`),
a parent element with the `inverted` class will switch the text to tints of white.

<Story name="Color Modifiers">
  <TableLayout 
    isCSS
    headers={['class', 'description', 'color', 'inverted']}
    rows={[
      ['color--heading', 'Heading text color'],
      ['color--primary', 'Primary text color (default)'],
      ['color--secondary', 'Secondary text color'],
      ['color--hint', 'Hint text color'],
      ['color--success', 'Success text color'],
      ['color--error', 'Error text color'],
    ].map(row => [
      row[0],
      row[1],
      { children: "Aa", className: row[0] },
      { className: "inverted", style: { backgroundColor: FDS.COLOR_NAVY }, children: <div className={row[0]}>Aa</div> }
    ])}
  />
</Story>
