## Child Align
Positions direct child(s) of element using flex properties.
**These classes modify `flex` properties.**
Avoid combining `alignChild` classes with flexbox layout elements.

Signature: `alignChild--(x-axis)--(y-axis)`

## Vertical Align
Sets `vertical-align` property. Note that `vertical-align`
**only applies to elements displayed `inline`, `inline-block`,
`table-cell` or `inline-flex`**.
